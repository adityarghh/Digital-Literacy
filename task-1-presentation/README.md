# Task 1 – Digital Literacy Awareness Infographic

## Tool Used
Canva (canva.com)

## Topics Covered
1. What Digital Literacy Is
2. Safe Internet Practices
3. Email Etiquette
4. Useful Digital Tools for Students

## Description
Created a one page infographic for first-year B.Tech students using Canva. 
It covers what digital literacy means, safe internet habits like using strong 
passwords and avoiding phishing, email etiquette do's and don'ts, and useful 
digital tools like Google Docs, Notion, LinkedIn, and Kaggle.
